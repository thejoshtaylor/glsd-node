<objective>
Capture token usage and context percentage from Claude result events into Process fields during Stream(), then read them in processMessage() and write to Session fields so /status displays real data.

Purpose: SESS-06 and SESS-07 require /status to show actual token counts and context percentage. The fields and accessors on Session already exist and buildStatusText already reads them -- the gap is that processMessage() never writes them because Process does not capture the data from result events.

Output: Process.LastUsage(), Process.LastContextPercent() accessors populated during Stream(); processMessage() writes to Session fields on success only.
</objective>
