   `type ChannelRateLimiter struct`:
   - `mu sync.Mutex`
   - `limiters map[int64]*rate.Limiter`
   - `limit rate.Limit`
   - `burst int`

   `func NewChannelRateLimiter(requestsPerWindow int, windowSeconds int) *ChannelRateLimiter`:
   - Compute `r := rate.Limit(float64(requestsPerWindow) / float64(windowSeconds))`
   - Return struct with `limiters: make(map[int64]*rate.Limiter)`, `limit: r`, `burst: requestsPerWindow`

   `func (crl *ChannelRateLimiter) Allow(channelID int64) (bool, time.Duration)`:
   - Lock mutex
   - Get or create `rate.NewLimiter(crl.limit, crl.burst)` for channelID
   - Unlock mutex
   - Call `l.Reserve()`
   - If `!r.OK()`: return `false, 0`
   - If `r.Delay() > 0`: call `r.Cancel()`, return `false, r.Delay()`
   - Return `true, 0`

   - `TestRateLimiterConcurrent`: 10 goroutines calling Allow on same channel — no panic, no race. Run with `-race`.

   `func ValidatePath(path string, allowedPaths []string) bool`:
   - Normalize: `cleanPath := filepath.Clean(path)`
   - Convert to forward slashes for consistent comparison: `cleanPath = filepath.ToSlash(cleanPath)`
   - For each allowed path: normalize same way, check `strings.HasPrefix(cleanPath, allowedPath)`
   - Return true if any prefix matches, false otherwise

   `func CheckCommandSafety(text string, blockedPatterns []string) (bool, string)`:
   - Convert text to lowercase: `lower := strings.ToLower(text)`
   - For each pattern in blockedPatterns: if `strings.Contains(lower, strings.ToLower(pattern))`: return `false, pattern`
   - Return `true, ""`

   - Return false
