**Dispatcher struct:**
```go
package dispatch

type Dispatcher struct {
    conn    ConnectionSender   // interface for testability
    cfg     *config.Config
    nodeCfg *config.NodeConfig
    log     zerolog.Logger
    audit   *audit.Logger
    limiter *security.ProjectRateLimiter

    mu        sync.RWMutex
    instances map[string]*instanceState

    wg     sync.WaitGroup
    stopCh chan struct{}
}

type instanceState struct {
    instanceID string
    project    string
    sessionID  string
    cancel     context.CancelFunc
    startedAt  time.Time
    done       sync.Once // gates terminal event (InstanceFinished or InstanceError)
}
```

**ConnectionSender interface** (for testability — ConnectionManager implements this):
```go
type ConnectionSender interface {
    Send(data []byte) error
    Receive() <-chan *protocol.Envelope
}
```

**Constructor:**
```go
func New(conn ConnectionSender, cfg *config.Config, nodeCfg *config.NodeConfig, auditLog *audit.Logger, limiter *security.ProjectRateLimiter, log zerolog.Logger) *Dispatcher
```

Initialize `instances` map, `stopCh` channel.

**Run(ctx context.Context):**
Loop reading from `d.conn.Receive()`, select on `ctx.Done()` and `d.stopCh` for exit. For each received envelope, call `d.dispatch(ctx, env)`.

**dispatch(ctx, env):**
1. Audit the inbound command: create `audit.NewEvent(env.Type, env.ID, d.nodeCfg.NodeID)` with InstanceID/Project populated from payload if decodable. Call `d.audit.Log(evt)`. Use structured zerolog: `d.log.Info().Str("type", env.Type).Str("msg_id", env.ID).Msg("received command")`.
2. Switch on `env.Type`:
   - `protocol.TypeExecute` -> `d.handleExecute(ctx, env)`
   - `protocol.TypeKill` -> `d.handleKill(env)`
   - `protocol.TypeStatusRequest` -> `d.handleStatusRequest(env)`
   - default -> `d.log.Warn().Str("type", env.Type).Msg("unknown command type")`

**handleExecute(ctx, env):**
1. Decode `protocol.ExecuteCmd` from envelope.
2. Check rate limit: `d.limiter.Allow(cmd.Project)`. If `cfg.RateLimitEnabled` is true and Allow returns false, send `InstanceError{InstanceID: cmd.InstanceID, Error: "rate limited"}` and return.
3. Register instance in map BEFORE spawning goroutine (prevents kill race):
   ```go
   instCtx, cancel := context.WithCancel(ctx)
   d.mu.Lock()
   d.instances[cmd.InstanceID] = &instanceState{
       instanceID: cmd.InstanceID,
       project:    cmd.Project,
       sessionID:  cmd.SessionID,
       cancel:     cancel,
       startedAt:  time.Now(),
   }
   d.mu.Unlock()
   ```
4. Send ACK using env.ID for correlation: `d.sendEnvelope(protocol.TypeACK, env.ID, protocol.ACK{InstanceID: cmd.InstanceID})`.
5. Spawn goroutine: `d.wg.Add(1); go d.runInstance(instCtx, cmd)`.

**runInstance(ctx, cmd):**
1. `defer d.wg.Done()`
2. `defer d.removeInstance(cmd.InstanceID)`
3. Get instance state for `done` Once reference.
4. Send `InstanceStarted{InstanceID, Project, SessionID}`.
5. Create zerolog sub-logger with node_id, instance_id, project fields (NODE-05).
6. Build args: `claude.BuildArgs(cmd.SessionID, d.cfg.AllowedPaths, "", d.cfg.SafetyPrompt)` — SessionID presence triggers --resume (INST-07).
7. Call `claude.NewProcess(ctx, d.cfg.ClaudeCLIPath, cmd.WorkDir, cmd.Prompt, args, config.FilteredEnv())`.
8. If NewProcess errors, use `inst.done.Do()` to send `InstanceError` and return.
9. Stream: `proc.Stream(ctx, func(event claude.ClaudeEvent) error { ... })`. In the callback, marshal the ClaudeEvent to JSON, wrap in `StreamEvent{InstanceID, jsonString}`, send via `d.sendEnvelope()`.
10. After Stream returns, capture `proc.SessionID()` and update instanceState.sessionID under lock.
11. Use `inst.done.Do()` to send terminal event: if `streamErr != nil && ctx.Err() == nil` send `InstanceError`, else send `InstanceFinished{InstanceID, ExitCode: 0}`.

**sendEnvelope(msgType, msgID, payload) helper:**
1. `protocol.Encode(msgType, msgID, payload)` -> marshal envelope to JSON -> `d.conn.Send(data)`.
2. If msgID is empty, use `protocol.NewMsgID()`.
3. Log errors but do not propagate (send failures during shutdown are expected).

**removeInstance(instanceID):**
Lock `d.mu`, delete from `d.instances`, unlock.

**Stop():**
Close `d.stopCh`.

**Wait():**
`d.wg.Wait()` — blocks until all instance goroutines complete.

**CRITICAL pitfall avoidance:**
- NEVER hold d.mu while calling d.conn.Send(). Copy needed data under lock, release, then send.
- `sync.Once` on instanceState.done gates the terminal event so kill + natural completion don't double-emit.
- defer ordering: `defer d.removeInstance()` then `defer d.wg.Done()` — Go executes defers LIFO, so Done runs first, then removeInstance. Actually we want removeInstance to run first (so Wait doesn't return while instance is still in map). Use: `defer func() { d.removeInstance(cmd.InstanceID); d.wg.Done() }()` as a single defer.

**handleKill(env):**
1. Decode `protocol.KillCmd`.
2. `d.mu.RLock()`, look up instance, `d.mu.RUnlock()`.
3. If not found, log warning and return.
4. Call `inst.cancel()` — this cancels the context passed to `claude.NewProcess`, which triggers SIGKILL via `exec.CommandContext`. The runInstance goroutine will detect `ctx.Err() != nil` and emit InstanceError via `inst.done.Do()`.

**handleStatusRequest(env):**
1. `d.mu.RLock()`, build `[]protocol.InstanceSummary` from instances map, `d.mu.RUnlock()`.
2. Build `protocol.NodeRegister` with NodeID, Platform (runtime.GOOS), Version (protocol.Version), Projects (empty slice for now — will be wired in Plan 03), RunningInstances.
3. Send via `d.sendEnvelope(protocol.TypeNodeRegister, env.ID, reg)`.

    var env protocol.Envelope
